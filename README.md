## Ecommerce creado para Reactjs de Coderhouse

El ecommerce es de una tienda de componentes para armar una computadora.

La app fue hecha con:

### `React`
### `Bootstrap 5`
### `Sweetalert2`

## Arrancar la app

Para iniciar la app utilizar el siguiente comando en la consola:

### `npm run start`
